API 调用示例

  -H "Content-Type: application/json" \
  -d '{
    "msgtype": "markdown",
    "markdown": {
      "title": "消息标题",
      "text": "消息内容"
    }
  }'

添加频道

公开消息：所有用户可见；私人订阅：需要授权
仅对公开消息有效，用于用户端分类展示
支持 JPG、PNG、GIF 格式，上传后可裁切为正方形
保存频道后，在频道列表中点击"令牌管理"按钮来添加、删除推送令牌。
一个频道可以拥有多个令牌，方便不同的信息源推送到同一个频道。
